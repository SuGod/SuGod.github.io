macOS 环境下 自带有 svn,下面来说说这个svn 怎么用.(针对 客户端 ,服务端还没有实践过)

从本地把已有项目上传至SVN服务器(初始化)

//svn import [本地项目地址] [svn仓库地址] --username=账号 --password=密码 -m "注释"
svn import [localhost project url] [svn:url] --username=xxxxx --password=xxxxx -m "注释."

从SVN服务器端下载到本地


svn checkout [svn:url] --username=xxxxx --password=xxxxx [localhost project url]

提交/更新

cd project/
svn commit -m "注释说明"
svn update

查看svn仓库

svn list [svn:url] --username=xxxxx --password=xxxxx

括号中代表 命令的简写,如:

svn commit   ===     svn ci
svn checkout   ===     svn co
*注意

在输入地址时 精良不要出现特殊字符如()中文等.

() 括号在使用是需要转义成 %28 %29 否则报错.
