技术栈: React Redux React-Router Antd-mobile Webpack NodeJs Express MongoDB

开发前准备:

nodeJS MongoDB create-react-app的安装就不赘述了.

ES6语法参考 <<阮一峰-ECMAScript 6 入门>>

正式开始:

  1. 使用create-react-app 搭建 react全家桶(react redux router webpack antd-mobile …) 过程>>

    终端执行npm run eject

    (如果项目使用git)执行这里可能会遇到一个错误

    解决办法是:在.gitignore文件中添加/.idea过滤项.然后重新commit and push;我认为原因是弹出配置这个操作是不可逆的,所以fb的工程师很人性化的去检查了git是否有未提交的内容,如果只是像看看配置项,或者不想弹出配置项了,还可以从git上恢复..idea是webstorm生成的编辑器配置文件,应该是随时变化的,所以把.idea过滤掉就可以了.

  2. 安装各种插件

yarn add redux react-redux react-router-dom antd-mobile babel-plugin-import babel-plugin-transform-decorators-legacy axios

yarn add express mongoose --dev
  1. 配置及前后台联调.

接下来在package.json中配置 plugins部分.其中第一项为antd-mobile的按需加载,第二项为transform-decorators-legacy插件,使babel支持装饰器写法.

"babel": {
    "presets": [
      "react-app"
    ],
    "plugins": [
      [
        "import",
        {
          "libraryName": "antd-mobile",
          "style": "css"
        }
      ],
      "transform-decorators-legacy"
    ]
}

然后配置server,在项目根路径下新建 server文件夹,用于存放后端server的内容.src下是前台内容.

关于express 官方文档有详细介绍.

在server下新建server.js:

const express = require('express');

const app = express();

//监听8080端口启动server
const server = app.listen('8080',function(){
    let host = server.address().address,
        port = server.address().port;
    //启动成功后在控制台打印.
    console.log('Express app listening at http://%s:%s',host,port);
})

在 package.json添加proxy配置项. 表示把请求转发到http://localhost:8080.

 "proxy":"http://localhost:8080"

ok,到这一步就基本完成了前后台的配置.接下来进行测试:

启动server.js,这里推荐 nodemon 自动重启的工具,使用 nodemon来启动server.js;

nodemon server/server.js

同时启动 react

yarn start

server.js添加路由

app.get('/user',function (req,res) {
    res.json({
        name:'张三',
        age:18
    })
});

修改/src/App.js

import React from 'react';

        super(props);

        console.log('component did mount!');
    }
    handleClick(){
        axios.get('/user').then(res => {
            console.log(res);
            if(res.status === 200){
                this.setState({
                    user:res.data
                })
            }
        })
    }

    render() {
        return (
            

                
                

                    我叫{this.state.user.name} ,我今年{this.state.user.age}岁.

    }
}

export default App;
